Drop teacher photos in here with these exact filenames and they'll appear automatically:

- `will.jpeg` — used on the homepage and the Drums, Brass and Jazz pages
- `rose.jpeg` — used on the homepage and the Cello and Piano pages

Square-ish photos work best, centered on the face — the site crops them into a circle
automatically. See CONTENT-GUIDE.md for more on picking/cropping photos.

Until a file exists, the page falls back to showing the camera-icon placeholder, so it's
safe to leave these unset.
